HTML template which is also platform for making great apps

NIGHTLY is an App-like HTML5 template built on jQuery Mobile framework.

jQuery Mobile is ready-made framework for portable websites and apps.

Why NIGHTLY?

It was never been easier to built mobile website. No more random classes, no more strange tags! NIGHTLY is super-easy to use.

Features

  • Pre-coded HTML5 elements (tables, sliders, sidebars, forms etc.)
  • CSS3 transitions
  • 20 HTML pages
  • Lightweight code
  • Works on most mobile devices
  • Works on most tablets
  • You can make your own color scheme with ThemeRoller
  • Left and right sidebars
  • 400+ Font Awesome icons
  • Different color schemes
  • Fixed header/footer support
  • PhoneGap support
  • Device check script
  • Working contact form
  • and much more!

We PhoneGap

NIGHTLY isn't just HTML template, it's a tool! You can built awesome apps with Nightly + PhoneGap, it's super-fun, check PhoneGap official website for more informations.

PhoneGap is a free, open source framework that allows you to create mobile apps using standardized web APIs.
