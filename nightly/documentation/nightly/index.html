HTML

Nightly works with jQuery Mobile version 1.4 and above. If you want to use this template with all it's functions and features you will have to follow simple HTML layout.

You don't want to forget important libraries that must be included within <head> selection. Basic header that can be used with all nercessary plugins looks like this:

<head>
    <title>[page title]</title>
        <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" />

        <!-- jQueryMobileCSS - styling-->
        <link rel="stylesheet" href="css/jqeury.mobile.theme.min.css" />

        <!-- jQueryMobile icons -->
        <link rel="stylesheet" href="css/jquery.mobile.icons.min.css" />

        <!-- jQueryMobile core CSS -->
    <link rel="stylesheet" href="css/jquery.mobile.min.css" />

        <!-- NIGHTLY style -->
        <link rel="stylesheet" href="css/nightly.css" />

        <!-- Load Google Fonts -->
        <link href='http://fonts.googleapis.com/css?family=Source+Sans+Pro:300,400,600,700,900,300italic,400italic,600italic,700italic|Oswald:400,700' rel='stylesheet' type='text/css'>

        <!-- Load Font Awesome Icons -->
        <link rel="stylesheet" href="css/font-awesome.min.css">

        <!-- jQuery / jQueryMobile Scripts -->
        <script src="js/jquery-2.1.1.min.js"></script>
    <script src="js/jquery.mobile.min.js"></script>

</head>

The HTML <body> layout is the same throughout the template. Here's the general structure:

<body>
    <div data-role="page" data-theme='a'> 
        <div data-role="panel" data-display="push" id="sidebar" data-theme="a"></div>
        <div data-role="header" data-tap-toggle="false" data-theme='a'></div>   
        <div data-role="content"></div>
        <div data-role="footer" data-theme="none" data-border="false"></div>
    </div>
</body>

Data variables

Nightly inherited data variables from jQuery Mobile. Proper usage of data variables can be found in jQuery Mobile Docs. Basicaly it's something like this:

<button data-role="button" data-theme="b" data-corners="false">Test</button>

For our purposes there are few data variables to remember:

data-role [role/none]     (read jQuery Mobile Docs)
data-theme [A/B/C]        (change color scheme)
data-ajax [true/false]    (turn on/off AJAX page loading)
data-border [true/false]  (NEW: removes any border from element)
data-corners [true/false] (removes rounded corners)
data-shadow [true/false]  (removes shadows)

Color Schemes

Available Color Variants

Using variable data-theme you can switch between two different color variations for this template. Yellow (data-theme="a") and blue (data-theme="b"). Example below:

<button data-role="button" data-theme="b">jQM button theme B</button>

Theme Styling

If you want to change color schemes (variable data-theme) or add new (more than A/B) please follow jQuery Mobile instructions for making new or changing current color schemes. Use simple tool called ThemeRoller and create new or import current color theme into roller. Then edit the colors, save and owerwrite current color theme CSS file jquery.mobile.theme.min.css. This is on your own risk. If you want to take back native color schemes overwrite this file with original jquery.mobile.theme.min.css. You can found this file in your downloaded ZIP with template.

How to use Nightly class

First thing you have to do is disable jQuery Mobile native styling, to do that add variable data-role with value none into the markup like so:

<button data-role="none">button</button>

Then add Nightly class of your chosing like so:

<button data-role="none" class="nightly-button">button</button>

And this is all it takes to create Nightly styled element (in this case button).

Additional styling for Nightly class element

You can use data variable to change style of the Nightly element using catiable data-theme.

<button data-role="none" class="nightly-button" data-theme="a">button</button>

Result: the button is now yellow with white text instead of white button with black border and black text.

In case you want to edit some of the Nightly classes open file css/nightly.css and locate your class. This is example of .nightly-button class:

.nightly-button, .nightly-input {
    background: none;
    border: 1px solid #2f3948;
    color: #2f3948;
    border-radius: 2px;
    -moz-border-radius: 2px;
    -webkit-border-radius: 2px;
    padding: 6px 14px;
    text-decoration: none;
}

Theme Styling

If you want to change color scheme (variable data-theme) of Nightly class element locate data-theme CSS markup for each class and edit those values:

.nightly-button[data-theme=a] {
    background: #ebce24;
    border: none;
    color: #ffffff;
}

Device Detect

Use of this plugin is very simple. First you must load it to the page in header like so:

<!-- Device Detect Script -->
<script src="js/device.detect.js"></script>

Then you will have list of available functions:

deviceDetect.Android() – TRUE if Android
deviceDetect.BlackBerry() – TRUE if BlackBerry
deviceDetect.iOS() – TRUE if iOS
deviceDetect.any() – TRUE if mobile device
isRetina() – TRUE if Retina display
 
Open file devices.html for usage example.

Slder

Owl Carousel

Is jQuery plugin which is super-easy to use. First you need to load nercessary libraries:

<!-- Owl Carousel -->
<link rel="stylesheet" href="css/owl.carousel.css">
<link rel="stylesheet" href="css/owl.theme.css">
<script src="js/owl.carousel.min.js"></script>
Then create slider selection using unique ID, slider class owl-carousel and add additional Nightly styling class (optional) nigtly-inline-slider or nightly-inline-gallery.
 
<div data-role="none" id="slider" class="owl-carousel nightly-inline-slider">
    <div data-role="none" class="item nightly-text-box align-center">
        <h3 class="nightly-text-box-title">Text Box in Carousel</h3>
        <p>Nunc et urna ullamcorper tortor eleifend pretium sit amet vel dui. Integer blandit ipsum in iaculis. Nam ultricies lectus.</p> 
    </div> 
    <div class="item">
      <button data-role="none" class="nightly-circle"><i class='fa fa-align-center'></i></button>
      <h3>Swipe me</h3>
      <p>This slider works fine with any type of content.</p>
    </div>
    <div class='item nightly-alert nightly-alert-warning'>
    	<i class='fa fa-2x fa-warning fa-fw'></i>
    	<p>You can put here almost everything even warnings.</p>
    </div>              
</div>

Don't forget to enable slider through jQuery:

$("#slider").owlCarousel();
List of available options and usage examples on Owl Carousel plugin page.
